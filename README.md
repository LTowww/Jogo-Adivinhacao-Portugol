# 🎯 Jogo de Adivinhação

Projeto desenvolvido em **Portugol** com o objetivo de praticar lógica de programação, estruturas de repetição e organização de código.
Futuro porte para **Python**

---

## 📌 Sobre o projeto

Este é um jogo simples de adivinhação onde o jogador deve descobrir um número secreto gerado aleatoriamente pelo sistema.

O jogo oferece diferentes níveis de dificuldade e fornece dicas ao usuário durante as tentativas.

---

## 🚀 Funcionalidades

* 🎚️ Sistema de dificuldade:

  * Fácil (1 a 10)
  * Médio (1 a 50)
  * Difícil (1 a 100)

* 🔁 Loop de tentativas até acertar

* 📢 Dicas durante o jogo:

  * "É maior"
  * "É menor"

* 🔄 Opção de jogar novamente

* 🧹 Interface limpa com atualização de tela

---

## 🧠 Conceitos utilizados

* Estruturas condicionais (`se`, `senao`)
* Estruturas de repetição (`enquanto`)
* Funções
* Geração de números aleatórios
* Organização de código
* Interação com o usuário

---

## ▶️ Como executar

1. Abra o código no **Portugol Studio**
2. Execute o programa
3. Escolha a dificuldade
4. Tente adivinhar o número secreto 🎯
